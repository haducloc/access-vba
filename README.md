### Access VBA Framework
- Build scalable, reliable database applications using Access forms as the front end.
- VBA knowledge is required, but the framework is easy to use, easy to maintain, and easy to scale.
- The database storage can be any SQL database (SQL Server, MySQL, PostgreSQL, Oracle, etc.).
- **Note:** Access is used **only** as the front end. The database storage must not be Access to ensure scalability, reliability, performance, multi-user concurrency, and cloud readiness, and no local Access tables are required.
- 🚀 **Still like Access? This framework is for you.**
